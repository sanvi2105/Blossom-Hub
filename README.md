# Flower E-Commerce Website

## Overview
The Flower E-Commerce Website is a responsive, visually appealing website designed to showcase and sell flowers online. It features modern UI/UX design, product listings, a shopping cart system, customer reviews, and a contact form. This project demonstrates proficiency in **HTML, CSS, JavaScript, and basic front-end web development concepts**.

---

## Features

- **Home Section**: Attractive landing page with hero section showcasing fresh flowers.
- **About Section**: Highlights why customers should choose this platform.
- **Products Section**:
  - Displays flower products with images, names, prices, and “Add to Cart” functionality.
  - Supports interactive cart operations using JavaScript.
- **Icons/Services Section**: Highlights services like free delivery, secure payment, return policy, and offers.
- **Customer Reviews Section**: Displays customer feedback with star ratings.
- **Contact Section**: Includes a contact form for users to reach out.
- **Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices.
- **Footer Section**: Quick links, extra links, contact info, and locations.

---

## Technologies Used

- **HTML5** – Structuring the web pages.
- **CSS3** – Styling and layout.
- **JavaScript** – Add to Cart functionality and interactive elements.
- **Font Awesome** – Icons for navigation, cart, and other UI elements.
- **Responsive Design** – Ensuring the website looks great on all devices.
